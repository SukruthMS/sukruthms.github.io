Financial Portfolio Tracker

Project information

MERN Stack
Oct 2022 - Dec 2022
Github

  • This project is a robust and scalable backend system developed with Node.js, designed to support a financial portfolio tracking application. The architecture leverages the CQRS (Command Query Responsibility Segregation) pattern to efficiently handle the separation of command and query operations, improving overall system performance. Using a Gateway design, the system ensures smooth routing and processing of requests, while real-time integration with financial data APIs provides up-to-date asset information. The backend also utilizes MongoDB's aggregation framework to handle complex data computations for over 100 tracked assets.


  • In addition to performance optimizations, the backend includes role-based access control, allowing secure access for multiple user roles such as Guest, Registered User, Financial Advisor, and Admin. Designed with scalability in mind, the system is capable of handling increased demand through horizontal scaling with load balancers. This backend is a demonstration of how thoughtful architecture can enhance both scalability and performance in modern financial applications.
